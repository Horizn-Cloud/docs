---
title: "List articles"
openapi: "GET /articles"
description: "List all articles for this site"
---

Returns a paginated list of articles. **Scope required:** `articles:read`

## Query parameters

<ParamField query="limit" type="integer" default="25">Results per page (1-100).</ParamField>
<ParamField query="cursor" type="string">Pagination cursor.</ParamField>
<ParamField query="article_type" type="string">Filter by article type.</ParamField>
