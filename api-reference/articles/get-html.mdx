---
title: "Get article HTML"
openapi: "GET /articles/{id}/html"
description: "Get the rendered HTML body of an article"
---

Returns the article's main body content as HTML. Send `Accept: text/html` to receive raw HTML instead of JSON.

**Scope required:** `articles:read`

## Path parameters

<ParamField path="id" type="string" required>UUID of the article.</ParamField>

<ResponseExample>
```json 200
{
  "data": {
    "id": "art-uuid-123",
    "html": "<h2>2025 Honda CR-V: A Comprehensive Review</h2><p>The 2025 Honda CR-V continues to...</p>",
    "component_status": "Complete",
    "updated_at": "2026-02-10T12:15:00.000Z"
  }
}
```
</ResponseExample>

<Tip>
  To get raw HTML directly (useful for embedding), set the `Accept` header to `text/html`.
</Tip>
