---
title: "Get IdeaCloud"
openapi: "GET /ideaclouds/{id}"
description: "Get details and status of a specific IdeaCloud"
---

Returns the full details of an IdeaCloud including its status, title, synopsis, tags, and recommended question IDs.

**Scope required:** `ideaclouds:read`

## Path parameters

<ParamField path="id" type="string" required>
  UUID of the IdeaCloud.
</ParamField>

<ResponseExample>
```json 200
{
  "data": {
    "id": "a1b2c3d4-...",
    "keyword": "2025 Honda CR-V reviews",
    "title": "Research: 2025 Honda CR-V reviews",
    "synopsis": "Exploring consumer questions about the 2025 Honda CR-V",
    "tags": ["honda", "cr-v", "suv", "2025"],
    "status": "complete",
    "category_id": null,
    "recommended_question_ids": ["q1-uuid", "q2-uuid", "q3-uuid"],
    "created_at": "2026-02-10T12:00:00.000Z",
    "updated_at": "2026-02-10T12:05:00.000Z"
  }
}
```
</ResponseExample>
