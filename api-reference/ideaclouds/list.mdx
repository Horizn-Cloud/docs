---
title: "List IdeaClouds"
openapi: "GET /ideaclouds"
description: "List all IdeaClouds for this site"
---

Returns a paginated list of IdeaClouds for the site associated with your API key.

**Scope required:** `ideaclouds:read`

## Query parameters

<ParamField query="limit" type="integer" default="25">
  Results per page (1-100).
</ParamField>

<ParamField query="cursor" type="string">
  Pagination cursor from a previous response.
</ParamField>

<ParamField query="status" type="string">
  Filter by status: `researching`, `complete`, `failed`.
</ParamField>

<ParamField query="keyword" type="string">
  Search by keyword (case-insensitive partial match).
</ParamField>
