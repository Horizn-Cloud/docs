---
title: "Get webhook deliveries"
openapi: "GET /webhooks/{id}/deliveries"
description: "View delivery log for a webhook subscription"
---

Returns the recent delivery attempts for a webhook subscription, including status codes and response times. Useful for debugging.

**Scope required:** `webhooks:read`

## Path parameters

<ParamField path="id" type="string" required>UUID of the webhook subscription.</ParamField>

## Query parameters

<ParamField query="limit" type="integer" default="25">Results per page (1-100).</ParamField>
<ParamField query="cursor" type="string">Pagination cursor.</ParamField>
