---
title: "List webhooks"
openapi: "GET /webhooks"
description: "List all webhook subscriptions for this site"
---

Returns all webhook subscriptions. **Scope required:** `webhooks:read`

<ResponseExample>
```json 200
{
  "data": [
    {
      "id": "wh-uuid-123",
      "url": "https://your-app.com/webhooks/horizn",
      "events": ["article.completed", "ideacloud.completed"],
      "description": "Content pipeline webhook",
      "is_active": true,
      "failure_count": 0,
      "created_at": "2026-02-10T12:00:00.000Z"
    }
  ]
}
```
</ResponseExample>
