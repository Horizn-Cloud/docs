---
title: "List inventory"
openapi: "GET /inventory"
description: "List vehicles in this site's inventory"
---

Returns a paginated list of vehicles from the dealership's current inventory.

**Scope required:** `inventory:read`

## Query parameters

<ParamField query="limit" type="integer" default="25">Results per page (1-100).</ParamField>
<ParamField query="cursor" type="string">Pagination cursor.</ParamField>
<ParamField query="make" type="string">Filter by vehicle make (e.g., `Honda`).</ParamField>
<ParamField query="model" type="string">Filter by vehicle model (e.g., `CR-V`).</ParamField>
<ParamField query="year" type="integer">Filter by model year.</ParamField>
<ParamField query="condition" type="string">Filter by condition: `new` or `used`.</ParamField>
