title: "articles:read"
description: "List and retrieve articles, HTML content, components, and compliance reports"
---

<code>articles:read</code>

The `articles:read` scope grants read-only access to articles and their associated data. This includes listing articles, viewing individual article details, retrieving rendered HTML, accessing component data, and viewing compliance reports.

## Endpoints

| Method | Path | Description |
| --- | --- | --- |
| `GET` | `/public/articles` | List articles |
| `GET` | `/public/articles/{id}` | Get article details |
| `GET` | `/public/articles/{id}/html` | Get rendered HTML |
| `GET` | `/public/articles/{id}/components` | Get all components |
| `GET` | `/public/articles/{id}/compliance` | Get compliance report |

---

## List articles

```
GET /public/articles
```

Returns a paginated list of articles for your site.

### Query parameters

| Name | Type | Default | Description |
| --- | --- | --- | --- |
| `limit` | integer | `25` | Number of results to return (1-100) |
| `cursor` | string | — | Base64-encoded cursor from a previous response |
| `article_type` | string | — | Filter by article type (e.g. `basic`, `qa`, `expert`) |

### Example

<CodeGroup>
```bash cURL
curl "https://public-api.hrizn.io/public/articles?limit=10&article_type=basic" \
  -H "X-API-Key: hzk_your_key_here"
```

```typescript TypeScript
const res = await fetch(
  "https://public-api.hrizn.io/public/articles?limit=10&article_type=basic",
  { headers: { "X-API-Key": "hzk_your_key_here" } }
);
const data = await res.json();
```

```python Python
import requests

response = requests.get(
    "https://public-api.hrizn.io/public/articles",
    headers={"X-API-Key": "hzk_your_key_here"},
    params={"limit": 10, "article_type": "basic"},
)
data = response.json()
```

```php PHP
$query = http_build_query(["limit" => 10, "article_type" => "basic"]);
$ch = curl_init("https://public-api.hrizn.io/public/articles?" . $query);
curl_setopt_array($ch, [
    CURLOPT_HTTPHEADER => ["X-API-Key: hzk_your_key_here"],
    CURLOPT_RETURNTRANSFER => true,
]);
$data = json_decode(curl_exec($ch));
```
</CodeGroup>

## Get all components

```
GET /public/articles/{id}/components
```

Returns all generated components for an article, or a single component if a type is specified.

### Path parameters

| Name | Type | Required | Description |
| --- | --- | --- | --- |
| `id` | string (UUID) | Yes | Article ID |

To retrieve a single component, append the type to the path:

```
GET /public/articles/{id}/components/{type}
```

| Name | Type | Required | Description |
| --- | --- | --- | --- |
| `type` | string | No | Component type (e.g. `Body`, `SeoMetadata`, `ArticleSchema`) |

See [Component Types](/api-reference/reference/component-types) for all available types.

## Error response — Missing scope

If your API key does not include `articles:read`, any request to these endpoints returns:

```json
{
  "error": {
    "code": "forbidden",
    "message": "This API key requires one of the following scopes: articles:read",
    "details": {
      "required_scopes": ["articles:read"]
    }
  }
}
```
